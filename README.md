# ts-tron

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.4. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

actuator : 트랜잭션 관련 디렉토리
chainbase : 기본 블록 구조
common: 공용 유틸리티, DB
consensus : 합의 알고리즘
crypto: 암호화
framework : 통신관련
